Service Worker

  • JavaScript file
  • Always running in background
  • In parallel to the UI thread
  • Acts as a proxy
  • Can't access the DOM
  • Async

Service Worker

Service Worker Registration

if(navigator.serviceWorker) { 
 navigator.serviceWorker.register("sw.js")
    .then(function (registration) {
        // here you can : 
        // define a scope, subscribe to push notifs...
       console.log('Yay!');
    }).catch("Service Worker Error");
}
